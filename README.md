# Discord Joke Bot

## ✨ Introduction
The perfect discord bot for your server. Fill your discord with jokes.

## 📝 Dependencies
- npm/yarn

### Install all dependencies
```
npm install
```

## 🚀 Build System

To run the bot
```
node src/index.js
```

To test the bot
```
npm test
```
